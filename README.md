# Song Tracker

Track songs you want to play next on your musical instrument. This is an app with web intreface for selecting songs from the MusicBrainz API via an id. You can add songs to completed/tracked to track/ mark as completed. You can create a new account to track songs and view other user profiles and their tracked songs.

## Features

1. User Authentication: Register and log in to your account.
2. Song Tracking: Add songs to your tracked or completed list.
3. Search: Search for songs, artists, and albums using the MusicBrainz API.
4. User Profiles: View other users' profiles and their tracked songs.
5. Tab2Piano Integration: Easily convert tabs to piano notes and back.

## Instalation

1. Press the blue button `<> Code`
2. Hover over the `Download Zip` button and click it to download the zip version of this repository

## Usage
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the application:
   ```bash
   npm start
   ```
3. Open your web browser and navigate to `http://localhost:3000`.
